# Changelog

Notable user-facing impact of **chain-hub** (CLI + bundled `core/`). Internal work (chores, refactor, docs, tests, CI, registry-sync) is not included.

## Pre-1.0 (beta)

Versions **below 1.0.0** may change CLI behaviour, defaults, and bundled `core/` layout without a full semver story. This file may stay minimal until **1.0.0**; use `npm show chain-hub version`, git history, or release tags for finer detail during beta. From **1.0.0** onward, run `bun run changelog` in `cli/` before publishing so user-facing changes are listed here.
